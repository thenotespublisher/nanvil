Everything you need for Neo3 dev

Local chain, dev RPC cheats, mainnet forks, and a built-in explorer — powered by nanvil.

▣

Local chain

Single-validator in-memory Neo3 network with auto-mining on every submitted transaction.

◎

Prefunded accounts

Configurable dev mnemonic and GAS balances — keys printed on startup and queryable via RPC.

⇄

Block explorer

Web UI on port 8546 with live WebSocket feed, contract deploys, and embedded documentation.

⎇

Mainnet / testnet forks

Lazy StateService fork at any block height — deploy and test against real remote state locally.

⚡

Dev RPC cheats

nanvil_mine, time travel, snapshots, mempool control, and evm_* aliases.

Cast-style CLI

ncast

Talk to a running node over JSON-RPC — send GAS, call contracts, deploy NEF files, inspect blocks, and stream live events. The Foundry cast equivalent for Neo3.

  • Balances, transfers, and NEP-17 send
  • Read-only and signed contract invocations
  • Deploy with --nef and --manifest
  • Raw RPC, mempool, and WebSocket watch
CLI reference Examples

Quick commands

export NCAST_RPC=http://127.0.0.1:8545

ncast balance <address>
ncast send --wif <wif> <to> 1
ncast call gas decimals

ncast deploy --wif <wif> \
  --nef contract.nef \
  --manifest contract.manifest.json

ncast watch

Multi-language compiler

nsmith

Compile Neo N3 smart contracts to .nef and .manifest.json artifacts ready for ncast deploy. The Foundry forge equivalent — one CLI for Go, Python, C#, and Java.

  • Go — embedded neo-go compiler, no extra install
  • Python — neo3-boa via managed toolchain
  • C# — Neo.Compiler.CSharp (nccs)
  • Java — neow3j Gradle plugin
nsmith guide Compile examples

Quick commands

# Go — works out of the box
nsmith compile --lang go ./main.go --out contract

# Python / C# / Java — install toolchains once
nsmith install --lang python,csharp
nsmith compile --lang python ./contract.py

nsmith init MyToken --lang go
nsmith doctor --all

The Nanvil stack

Three binaries, one release — modeled after Foundry’s Anvil, Cast, and Forge.

nanvil

↔ Anvil

Local Neo3 node with prefunded accounts, forks, and dev RPC cheats.

ncast

↔ Cast

RPC client for transactions, calls, deploys, and chain inspection.

nsmith

↔ Forge

Multi-language contract compiler producing NEF + manifest artifacts.
